# AGENTS.md

Guidance for Codex (or any non-Claude agent) working in this repository.

**This file is deliberately a stub.** It used to duplicate CLAUDE.md's architecture guide and rotted several eras out of date — a stale wiring checklist is worse than none. There is one source of truth:

1. **Read `STATUS.md` (repo root) first** — the live current-state brief: which branch the work is on, what just landed, what's next. Then cross-check `git status` + `git log --oneline -5`; if they disagree with STATUS.md, fix STATUS.md before building.
2. **Read `CLAUDE.md` (repo root) for everything else** — architecture, wiring checklists, conventions, gotchas, key-files table. Follow its "Keeping CLAUDE.md current" and "Keeping STATUS.md current" rules; update those files, not this one.
